# Rust e-Boekhouden

Rust OpenAPI client for [e-Boekhouden](https://www.e-boekhouden.nl/).

## Installation

```shell
cargo add eboekhouden
```

## Documentation

Documentation for is available on [Docs.rs](https://docs.rs/eboekhouden/latest/eboekhouden/).
